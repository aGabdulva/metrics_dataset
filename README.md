# metrics_dataset

В данном проекте мною реализована витрина данных(датасет в Redash) для дальнейшей реализации выскороуревного дашборда для CPO.
Для реализации витрины был написан SQL-скрипт, который можно увидеть в файле `script`.

**Описание данных**

Данные лежат в табличках

public_aaa.aaa_bi_items
public_aaa.aaa_bi_dict_item_categories
public_aaa.aaa_bi_item_price
public_aaa.aaa_bi_orders
public_aaa.aaa_bi_order_delivery_time
public_aaa.aaa_bi_order_items
public_aaa.aaa_bi_customers

Описание таблиц и полей можно найти в файле `description`

Примечание: скриншоты витрины не включены в данный проект, так как вставлять скриншоты витрины из Redash запрещено ввиду NDA.
